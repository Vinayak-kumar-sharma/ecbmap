# 📍 Campus Navigator

> A smart and intuitive way to explore your campus — built with a focus on clarity, usability, and real-world navigation problems.

---

## 📖 Overview

This **Campus Navigator** project is designed and developed under the guidance of our coordinator to simplify campus navigation for students and visitors.

Navigating large campuses can often be confusing — especially when searching for classrooms, labs, or facilities across different floors. This project was built to solve that exact problem by providing a **clear, interactive, and user-friendly navigation system**.

Instead of relying entirely on external tools, we created **custom indoor and outdoor maps using SVG**, allowing precise control over layout, structure, and interaction.

---

## ✨ Features

- 🗺️ Outdoor + Indoor Navigation  
- 🏢 Floor-wise Maps (Main Floor, First Floor)  
- 🔍 Smart Search with Suggestions  
- 🧭 Start → Destination Navigation  
- 📱 Fully Responsive (Mobile + Desktop)  
- 🔄 Map Switching  
- 🔎 Zoom Controls  
- ⚡ Clean and Simple UI  

---

## 🧠 Design Approach

The core goal of this project is **user experience**.

We focused on:
- Making navigation simple and intuitive  
- Reducing confusion through clear structure  
- Improving discoverability of locations  
- Ensuring usability across all devices  

Special attention was given to handling **inconsistent naming of locations**, which is a common real-world problem. Instead of relying only on exact matches, the system is designed to guide users through better search and interaction patterns.

## 🚀 How to Use

1. Open the project in your browser  
2. Select a map (Outdoor / Floor)  
3. Use the search bar to find a location  
4. Enter start and destination for navigation  
5. Use zoom controls for better view
